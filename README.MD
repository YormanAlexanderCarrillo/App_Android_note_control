### Clonar el repositorio 

```bash
$ git clone https://github.com/YormanAlexanderCarrillo/App_Android_note_control.git
```

### ingreso a la carpeta del proyecto
```bash
$ cd App_Android_note_control
```

### Instalar modulos 
```bash
$ npm install
```

### Ejecutar en modo desarrollador
```bash
$ npx expo start
```
